# DevFlow for Cursor

Integration guide for using DevFlow skills with Cursor.

## Installation

DevFlow skills are automatically available in Cursor when placed in `.skills/devflow/`.

## Usage

Cursor understands natural language:

```
"开始工作"  → standup
"下班"      → wrap
"分析需求"  → spec
```
